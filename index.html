Instructions

1. Download both the Curseforge installer and the modpack profile.

2. Install the Curseforge client. Once done, select the Minecraft icon/selection. Run through the installation steps.

3. Press the "Create Custom Profile" option at the top right of the screen. Click the import link.

4. Select the modpack profile (.zip) you downloaded when met with the file explorer window. Once selected, click the "Create" button.

5. You can connect to the server using blamejenny.ddns.net.
Please Note: If you have a mod or version mismatch when attempting to connect, please make sure you have the most up to date version of the pack!
